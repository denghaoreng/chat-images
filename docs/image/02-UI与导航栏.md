# UI 与导航栏 — 顶部按钮和弹窗面板

## 概述

聊天图片插件需要在 SillyTavern 的**顶部导航栏**添加一个图标按钮，点击后打开功能面板（弹窗），再次点击则关闭。

SillyTavern 的导航栏使用 **drawer（抽屉）** 模式：导航栏中的图标按钮和对应的内容面板通过 `drawer` 结构组织在一起。

---

## 导航栏 Drawer 结构

SillyTavern 的顶部导航栏定义在 `public/index.html` 中，结构如下：

```html
<!-- 导航栏图标区域 -->
<div id="top-bar">
</div>
<!-- 导航栏面板容器 -->
<div id="top-settings-holder">
    <!-- AI 配置抽屉 -->
    <div id="ai-config-button" class="drawer">
        <div class="drawer-toggle drawer-header">
            <div class="drawer-icon fa-solid fa-sliders fa-fw closedIcon" title="AI Response Configuration"></div>
        </div>
        <div class="drawer-content fillLeft closedDrawer">
            <!-- 面板内容 -->
        </div>
    </div>
    <!-- 世界书抽屉 -->
    <div id="WI-SP-button" class="drawer">
        <div class="drawer-toggle drawer-header">
            <div class="drawer-icon fa-solid fa-book-atlas fa-fw closedIcon" title="World Info"></div>
        </div>
        <div class="drawer-content closedDrawer">
            <!-- 面板内容 -->
        </div>
    </div>
    <!-- ... 更多抽屉 -->
</div>
```

### 关键 CSS 类

| CSS 类 | 用途 |
|--------|------|
| `.drawer` | 抽屉容器，包含图标和面板 |
| `.drawer-toggle` / `.drawer-header` | 可点击的图标区域 |
| `.drawer-icon` | 图标元素 |
| `.drawer-content` | 面板内容区域 |
| `.closedIcon` / `.openIcon` | 图标状态（关闭/打开） |
| `.closedDrawer` / `.openDrawer` | 面板状态（关闭/打开） |
| `.fillLeft` | 面板从左侧展开（可选） |

### 切换机制

SillyTavern 内置的 `doNavbarIconClick()` 函数处理抽屉的展开/收起：

```javascript
// 当点击 .drawer-toggle 元素时触发
// 1. 关闭其他已打开的抽屉
// 2. 切换当前抽屉的打开/关闭状态
// 3. 切换图标的 openIcon/closedIcon 状态
```

## 聊天图片插件的 Drawer 实现

聊天图片插件需要在初始化时动态创建自己的 Drawer 并添加到 `#top-settings-holder` 中。

### HTML 结构

```html
<div id="chat-images-drawer" class="drawer">
    <div class="drawer-toggle drawer-header">
        <div id="chatImagesDrawerIcon" class="drawer-icon fa-solid fa-image fa-fw closedIcon" title="聊天图片"></div>
    </div>
    <div id="chat-images-panel" class="drawer-content closedDrawer">
        <div class="drawer-content-inner">
            <!-- 功能面板内容 -->
            <div class="chat-images-panel-header">
                <h3>聊天图片设置</h3>
            </div>
            <div class="chat-images-panel-body">
                <!-- 正则规则列表 -->
                <div id="chat-images-rule-list">
                    <!-- 动态渲染 -->
                </div>
                <!-- 添加规则按钮 -->
                <button id="chat-images-add-rule" class="menu_button">
                    <i class="fa-solid fa-plus"></i> 添加规则
                </button>
            </div>
        </div>
    </div>
</div>
```

### 在 init 中动态添加

```javascript
// index.js
import { getContext } from '../../../extensions.js';

const MODULE_NAME = 'chat-images';

export async function init() {
    const html = await renderExtensionTemplateAsync(MODULE_NAME, 'settings', {});
    $('#extensions_settings').append(html);

    // 同时创建顶栏抽屉按钮
    addNavBarDrawer();
}

function addNavBarDrawer() {
    const drawerHtml = `
    <div id="chat-images-drawer" class="drawer">
        <div class="drawer-toggle drawer-header">
            <div id="chatImagesDrawerIcon" class="drawer-icon fa-solid fa-image fa-fw closedIcon" title="聊天图片"></div>
        </div>
        <div id="chat-images-panel" class="drawer-content closedDrawer">
            <div class="panelControlBar flex-container">
                <div class="fa-solid fa-grip drag-grabber hoverglow"></div>
                <div class="fa-solid fa-circle-xmark hoverglow dragClose" id="chatImagesDrawerClose"></div>
            </div>
            <div class="drawer-content-inner">
                <h3 class="margin0"><span>聊天图片</span></h3>
                <div id="chat-images-rule-list" class="margin5"></div>
                <div class="flex-container margin5">
                    <button id="chat-images-add-rule" class="menu_button">
                        <i class="fa-solid fa-plus"></i> 添加规则
                    </button>
                </div>
            </div>
        </div>
    </div>`;

    $('#top-settings-holder').append(drawerHtml);

    // 注册图标点击事件
    $('#chat-images-drawer .drawer-toggle').on('click', async function() {
        const { doNavbarIconClick } = await import('../../../script.js');
        doNavbarIconClick.call(this);
    });
}
```

## 交互流程

```
用户点击导航栏图标
        │
        ▼
   ┌─────────────────────────────┐
   │ doNavbarIconClick()         │
   │ 1. 关闭其他打开的抽屉       │
   │ 2. 切换 current drawer      │
   │ 3. 切换图标状态             │
   └─────────────────────────────┘
        │
        ▼
   ┌─────────────────────────────┐
   │ 功能面板显示/隐藏           │
   │ - 显示：加载规则列表数据    │
   │ - 隐藏：无需操作            │
   └─────────────────────────────┘
```

---

## 注意事项

1. **避免重复添加**: 在 `init` 中添加 drawer 前，先检查是否已存在
2. **事件清理**: 在 `onDisable` 或 `onDelete` 钩子中移除添加的 DOM 元素
3. **响应式适配**: 确保面板在移动设备上也能正常显示
4. **国际化**: 使用 `data-i18n` 属性支持多语言
