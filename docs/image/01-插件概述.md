# 聊天图片（Chat Images）插件 — 概述

## 什么是聊天图片插件？

聊天图片（Chat Images）是一个 SillyTavern 前端扩展插件，它能够在 AI 回复内容中根据正则表达式匹配，自动插入预设图片到聊天消息中，为聊天增添视觉表现力。

采用纯前端实现，已拆分为多模块架构，依赖 SillyTavern 内置 API，无需服务端插件。

### 核心功能

| 功能 | 说明 |
|------|------|
| **导航栏入口** | 顶部导航栏 Drawer 抽屉按钮，与酒馆原生UI风格一致 |
| **角色图片集** | 按角色分组管理规则集 |
| **规则集管理** | 规则的分组管理，可绑定到角色集 |
| **正则匹配** | AI 回复后自动检测，支持多条规则、多规则集顺序匹配 |
| **图片绑定** | 一个正则可绑定多张图片，每张图片可设置权重控制出现概率 |
| **批量上传** | 单规则批量上传 + 批量创建规则（一图一规则） |
| **加权随机展示** | 按权重概率选择图片，支持规则集批次队列和停留时间 |
| **图片放大** | 双击图片或点击放大镜按钮全屏查看 |
| **规则折叠记忆** | 折叠/展开状态持久化到设置 |
| **标签页记忆** | 关闭再打开后保留上次的标签页 |
| **展开/折叠全部** | 一键展开或折叠所有规则 |
| **正则手册** | 内置正则表达式语法参考 |

## 插件架构

聊天图片插件遵循 SillyTavern 标准前端扩展架构，已拆分为多模块：

```
third-party/chat-images/
├── manifest.json          # 插件清单（必需）
├── index.js               # 入口：生命周期、事件监听、消息处理
├── data.js                # 数据层：CRUD、设置管理
├── drawer.js              # 导航栏：Drawer 面板
├── rules-ui.js            # 规则界面：渲染、事件绑定
├── image.js               # 图片管理：上传、删除、放大
├── matcher.js             # 匹配引擎：正则匹配、队列
├── popups.js              # 弹窗：批量添加、正则手册
├── utils.js               # 工具函数
├── style.css              # 样式文件
├── settings.html          # 设置面板 HTML 模板
└── docs/                  # 文档目录
    ├── 01-插件概述.md ~ 09-Bug修复总结.md
    └── docs-main/         # SillyTavern 插件开发指南
```

### 模块职责

| 模块 | 职责 | 核心导出 |
|------|------|---------|
| `index.js` | 生命周期钩子、事件注册、消息回调 | `init`, `onInstall`, `onDelete` |
| `data.js` | 规则/规则集/角色集 CRUD、设置持久化 | `addRule`, `getRulesData`, `saveSettings` |
| `drawer.js` | 导航栏 Drawer 创建与交互 | `addNavBarDrawer` |
| `rules-ui.js` | 三个标签页的渲染与事件绑定 | `renderRuleList`, `renderRuleSetList`, `renderCharSetList` |
| `image.js` | 图片上传/删除/放大 | `handleImageUpload`, `chatImageEnlarge` |
| `matcher.js` | 正则匹配、加权随机、图片队列 | `performMatch`, `queueBatchesForMessage` |
| `popups.js` | 批量添加规则弹窗、正则手册 | `showBatchAddPopup`, `showRegexHelp` |
| `utils.js` | 通用工具函数 | `generateId`, `escapeHtml` |

### 技术栈

- **语言**: 原生 JavaScript（ES Module）
- **UI**: jQuery + 原生 DOM 操作
- **存储**: `extensionSettings`（通过 `getContext().saveSettingsDebounced()` 持久化）
- **弹窗**: SillyTavern 内置 `callGenericPopup` + `POPUP_TYPE`
- **文件上传**: SillyTavern 内置 `/api/files/upload` + `/api/files/delete`
- **样式**: CSS + Font Awesome 图标

## 安装方式

### 开发环境安装

1. 将插件放在 `public/scripts/extensions/third-party/chat-images/` 目录下
2. 重启 SillyTavern 服务端或刷新页面
3. 在扩展管理菜单中启用"聊天图片"插件

### 用户安装

通过 SillyTavern 的 "Download Extensions & Assets" 菜单安装。

---

## 依赖要求

- **SillyTavern 版本**: ≥ 1.0.0
- **无外部依赖**: 纯前端插件，不依赖 Extras API 或第三方服务
