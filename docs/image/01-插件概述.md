# 聊天图片（Chat Images）插件 — 概述

## 什么是聊天图片插件？

聊天图片（Chat Images）是一个 SillyTavern 前端扩展插件，它能够在 AI 回复内容中根据正则表达式匹配，自动弹出或插入预设图片，为聊天增添视觉表现力。

### 核心功能

| 功能 | 说明 |
|------|------|
| **正则匹配** | AI 回复后自动检测，用正则表达式匹配文本内容 |
| **图片绑定** | 一个正则可绑定多张图片，每张图片可设置权重控制出现概率 |
| **图片管理** | 用户上传图片，复制存储到插件文件夹，配套生成 JSON 元数据 |
| **多种展示方式** | 支持居中模态框弹窗展示 和/或 聊天消息中插入图片 |
| **导航栏入口** | 在酒馆顶部导航栏添加"聊天图片"图标按钮，点击展开/收起功能面板 |

---

## 插件架构

聊天图片插件遵循 SillyTavern 标准前端扩展架构：

```
third-party/chat-images/
├── manifest.json          # 插件清单（必需）
├── index.js               # 主 JS 入口（必需）
├── style.css              # 样式文件
├── settings.html          # 设置面板 HTML 模板
├── images/                # 用户上传的图片存储目录
├── data.json              # 正则规则与图片绑定数据存储
└── docs/                  # 插件文档
    ├── 01-插件概述.md
    ├── 02-UI与导航栏.md
    ├── 03-正则匹配系统.md
    ├── 04-图片管理.md
    ├── 05-配置与存储.md
    ├── 06-事件与触发.md
    └── 07-完整代码示例.md
```

### 技术栈

- **语言**: 原生 JavaScript（ES Module）
- **UI**: jQuery + 原生 DOM 操作
- **存储**: `extensionSettings`（全局设置）+ 独立 `data.json`（规则数据）
- **样式**: CSS + Font Awesome 图标

---

## 安装方式

### 开发环境安装

1. 将插件放在 `public/scripts/extensions/third-party/chat-images/` 目录下
2. 重启 SillyTavern 服务端或刷新页面
3. 在扩展管理菜单中启用"聊天图片"插件

### 用户安装

通过 SillyTavern 的 "Download Extensions & Assets" 菜单安装。

---

## 依赖要求

- **SillyTavern 版本**: ≥ 1.0.0
- **无外部依赖**: 纯前端插件，不依赖 Extras API 或第三方服务
