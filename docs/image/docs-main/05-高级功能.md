# 高级功能 — 斜杠命令、宏、拦截器、Action Loader 等

## 斜杠命令（Slash Commands）

### 新版方式（推荐）

使用 `SlashCommandParser.addCommandObject()` 注册斜杠命令：

```javascript
import { SlashCommandParser } from '../../../slash-commands/SlashCommandParser.js';
import { SlashCommand } from '../../../slash-commands/SlashCommand.js';
import { ARGUMENT_TYPE, SlashCommandArgument, SlashCommandNamedArgument } from '../../../slash-commands/SlashCommandArgument.js';
import { commonEnumProviders } from '../../../slash-commands/SlashCommandCommonEnumsProvider.js';

SlashCommandParser.addCommandObject(SlashCommand.fromProps({
    name: 'repeat',
    callback: (namedArgs, unnamedArgs) => {
        const times = Number(namedArgs.times ?? 5);
        const text = unnamedArgs.toString();
        const space = isTrueBoolean(namedArgs.space.toString());
        return Array(times).fill(text).join(space ? ' ' : '');
    },
    aliases: ['example-command'],
    returns: '重复后的文本',
    namedArgumentList: [
        SlashCommandNamedArgument.fromProps({
            name: 'times',
            description: '重复次数',
            typeList: [ARGUMENT_TYPE.NUMBER],
            defaultValue: '5',
        }),
        SlashCommandNamedArgument.fromProps({
            name: 'space',
            description: '是否用空格分隔',
            typeList: [ARGUMENT_TYPE.BOOLEAN],
            defaultValue: 'off',
            enumList: ['on', 'off'],
        }),
    ],
    unnamedArgumentList: [
        SlashCommandArgument.fromProps({
            description: '要重复的文本',
            typeList: [ARGUMENT_TYPE.STRING],
            isRequired: true,
        }),
    ],
    helpString: `
        <div>将提供的文本重复指定次数。</div>
        <div><strong>示例:</strong>
            <ul>
                <li><code>/repeat foo</code> — 返回 "foofoofoofoofoo"</li>
                <li><code>/repeat times=3 space=on bar</code> — 返回 "bar bar bar"</li>
            </ul>
        </div>
    `,
}));
```

```javascript
SlashCommandParser.addCommandObject(SlashCommand.fromProps({
```

## 自定义宏（Macros）

### 新版宏系统（推荐）

```javascript
const { macros } = SillyTavern.getContext();

// 简单宏
macros.register('tomorrow', {
    description: '返回明天的日期',
    handler: () => {
        return new Date(Date.now() + 86400000).toLocaleDateString();
    },
});

// 带参数和分类的宏
macros.register('greet', {
    description: '生成问候语',
    category: MacroCategory.UTILITY,
    unnamedArgs: [
        { name: 'name', description: '要问候的名字' },
    ],
    handler: ({ unnamedArgs }) => {
        const [name] = unnamedArgs;
        return `你好, ${name}!`;
    },
});

// 解注册宏
macros.registry.unregisterMacro('greet');

// 注册别名
macros.registerAlias('greet', 'hello', { visible: true });
```

> `MacroCategory` 从 `/scripts/macros/macro-system.js` 导入。

### 🔍 Memory 插件的宏

```javascript
// 注册 {{summary}} 宏 — 可在任何支持宏替换的地方使用
if (power_user.experimental_macro_engine) {
    macros.register('summary', {
        category: MacroCategory.CHAT,
        description: '返回当前聊天的最新记忆/总结。',
        handler: () => {
            const uiSummary = $('#memory_contents').val().toString();
            if (uiSummary.trim().length > 0) return uiSummary;
            return getLatestMemoryFromChat(getContext().chat);
        },
    });
}
```

## 提示词拦截器（Prompt Interceptors）

拦截器允许插件在文本生成请求发送前修改聊天数据、注入内容或中止生成。

### 注册拦截器

在 `manifest.json` 中配置：

```json
{
    "generate_interceptor": "myCustomInterceptorFunction"
}
```

### 实现拦截器函数

函数必须是全局的，通过 `globalThis` 暴露：

```javascript
globalThis.myCustomInterceptorFunction = async function(chat, contextSize, abort, type) {
    // chat: 聊天历史消息数组（可修改）
    // contextSize: 当前上下文大小（token 数）
    // abort: 中止函数，传 true 可阻止生成
    // type: 生成类型，如 'quiet', 'regenerate', 'impersonate', 'swipe'

    // 示例：在最后一条用户消息前添加系统注释
    const systemNote = {
        is_user: false,
        name: 'System Note',
        send_date: Date.now(),
        mes: '由扩展添加的内容',
    };
    chat.splice(chat.length - 1, 0, systemNote);
};
```

> 拦截器按 `loading_order` 顺序依次执行，数值小的先执行。

插件可以为 LLM 注册可供调用的函数工具：

```javascript
const { registerFunctionTool } = SillyTavern.getContext();

registerFunctionTool({

    parameters: {
        $schema: 'http://json-schema.org/draft-04/schema#',
        type: 'object',

            location: { type: 'string', description: '城市名' },
        },
        required: ['location'],
    },
    action: async ({ location }) => {

        return JSON.stringify(data);
    },
});
```

## 消息格式化钩子（Message Formatting Hooks）

> ⚠️ 此功能目前仅在 staging 分支可用。

允许插件在消息渲染到 DOM 之前修改文本。

### 注册钩子

```javascript
const { messageFormatter } = SillyTavern.getContext();

// 简单钩子
messageFormatter.addHook((mes, ctx) => {
    if (ctx.isUser) return mes;
    return addFurigana(mes);  // 添加注音
});

// 指定阶段和顺序
messageFormatter.addHook((mes, ctx) => {
    return mes.replace(/\*\*(.+?)\*\*/g, '<mark>$1</mark>');
}, {
    stage: messageFormatter.stage.AFTER_MARKDOWN,
    order: messageFormatter.order.EARLY,  // 0=Earliest, 10=Early, 50=Normal, 90=Late, 100=Latest
});
```

### 钩子上下文

```javascript
{
    characterName: string,  // 角色名
    isSystem: boolean,      // 是否系统消息
    isUser: boolean,        // 是否用户消息
    messageId: number,      // 消息索引（-1 为临时消息）
    isReasoning: boolean,   // 是否推理输出
    stage: string,          // 当前管道阶段
}
```

> 上下文对象被 `Object.freeze()` 冻结，不可修改。

### 处理管道顺序

```
1. Prompt-bias 剥离
2. 注释/隐藏消息标准化
3. → beforeRegex 扩展钩子
4. 自定义正则规则
5. → afterRegex 扩展钩子
6. Markdown 自动修复
7. HTML 标签编码
8. Showdown Markdown → HTML 转换
9. → afterMarkdown 扩展钩子（默认）
10. 名称前缀剥离
11. DOMPurify 净化
```

## Action Loader（操作加载器）

用于长时间运行操作的加载覆盖层和 Toast 通知系统。

```javascript
const { loader } = SillyTavern.getContext();

// 基本阻塞加载器
const handle = loader.show({ message: '处理数据中...' });
try {
    const result = await someExpensiveOperation();
} finally {
    await handle.hide();
}

// 非阻塞加载器（后台任务）
const handle2 = loader.show({
    blocking: false,
    message: '后台下载中...',
    onStop: () => abortDownload(),
});
```

### 选项

| 选项 | 默认值 | 说明 |
|------|--------|------|
| `blocking` | `true` | 显示全屏覆盖层 |
| `message` | `'Generating...'` | Toast 通知消息 |
| `title` | `''` | Toast 标题 |
| `toastMode` | `'stoppable'` | `'stoppable'`（带停止按钮）、`'static'`、`'none'` |
| `stopTooltip` | `'Stop'` | 停止按钮提示 |
| `onStop` | `null` | 自定义停止处理，默认调用 `stopGeneration()` |
| `onHide` | `null` | 加载器隐藏时回调 |
| `overlayContent` | `null` | 自定义覆盖层内容 |

> 多个加载器可同时存在，覆盖层在所有加载器都隐藏后才消失。

## 弹窗与用户反馈（Popups）

### 快捷弹窗

```javascript
const { Popup, POPUP_RESULT } = SillyTavern.getContext();

// 确认对话框
const confirmed = await Popup.show.confirm('确认操作', '确定继续吗？');
// 返回 POPUP_RESULT.AFFIRMATIVE 或 POPUP_RESULT.NEGATIVE

// 文本输入对话框
const userInput = await Popup.show.input('输入名字', '请输入名字：', '默认值');
// 返回输入字符串，取消时返回 null

// 信息展示
await Popup.show.text('信息', '操作完成。');
```

### 自定义弹窗

```javascript
const popup = new Popup(
    '<div>自定义 HTML 内容</div>',
    POPUP_TYPE.TEXT,
    '',
    {
        wide: true,
        okButton: '保存',
        cancelButton: '取消',
        customButtons: [
            { text: '导出', icon: 'fa-download', result: POPUP_RESULT.CUSTOM1 },
        ],
        allowVerticalScrolling: true,
    }
);
const result = await popup.show();
```

## 其他功能

### 数据采集器（Data Bank Scrapers）

```javascript
const { registerDataBankScraper } = SillyTavern.getContext();

await registerDataBankScraper({
    id: 'my_scraper',
    name: '我的数据源',
    description: '从我的数据源导入',
    iconClass: 'fa-solid fa-database',
    iconAvailable: true,
    isAvailable: async () => true,
    scrape: async () => {
        const content = await fetchDataFromSource();
        return [new File([content], 'data.txt', { type: 'text/plain' })];
    },
});
```

### 调试功能（Debug Functions）

```javascript
const { registerDebugFunction } = SillyTavern.getContext();

registerDebugFunction(
    'my_ext_clear_cache',
    '清理我的扩展缓存',
    '清除所有缓存数据',
    async () => {
        const { localforage } = SillyTavern.libs;
        await localforage.removeItem('my_extension_cache');
        toastr.success('缓存已清理');
    }
);
```

### 变量系统（Variables）

```javascript
const { variables } = SillyTavern.getContext();

// 局部变量（当前会话）
variables.local.set('myKey', 'value');
const val = variables.local.get('myKey');
variables.local.has('myKey');
variables.local.del('myKey');
variables.local.inc('counter');  // 递增
variables.local.dec('counter');  // 递减

// 全局变量（跨会话持久化）
variables.global.set('myGlobalKey', { data: 42 });
```
