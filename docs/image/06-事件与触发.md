# 事件与触发

## 概述

聊天图片插件的核心触发逻辑是：**AI 回复消息后**，自动检测消息文本并执行正则匹配。事件监听代码位于 `index.js` 中，匹配逻辑在 `matcher.js` 中，图片渲染在 `rules-ui.js` 中。

---

## 事件选择

| 事件 | 触发时机 | 用途 |
|------|----------|------|
| `MESSAGE_RECEIVED` | LLM 消息已生成，未渲染到 UI | ✅ 主要匹配时机 |
| `MESSAGE_SWIPED` | 用户滑动切换回复 | 切换回复时重新匹配 |
| `CHAT_CHANGED` | 切换聊天 | 清除旧图片队列 |

---

## 事件注册 (index.js)

```javascript
import { getContext } from '../../../extensions.js';

export function init() {
    loadSettings();

    const { eventSource, event_types } = getContext();
    eventSource.on(event_types.MESSAGE_RECEIVED, onMessageReceived);
    eventSource.on(event_types.MESSAGE_SWIPED, onMessageSwiped);
    eventSource.on(event_types.CHAT_CHANGED, onChatChanged);
}
```

## 消息处理逻辑

### 1. 消息接收处理器

```javascript
function onMessageReceived(data) {
    if (!currentSettings.enabled) return;
    if (data.is_user || data.is_system) return;
    const text = data.mes;
    if (!text) return;

    // 执行匹配（matcher.js）
    const results = performMatch(text);
    if (results.length > 0) {
        queueBatchesForMessage(results);
    }
}
```

### 2. 滑动切换处理器

```javascript
function onMessageSwiped() {
    if (!currentSettings.enabled) return;
    const { chat } = SillyTavern.getContext();
    const currentMessage = chat[chat.length - 1];
    if (!currentMessage || currentMessage.is_user) return;
    const results = performMatch(currentMessage.mes);
    if (results.length > 0) {
        queueBatchesForMessage(results);
    }
}
```

### 3. 聊天切换处理器

```javascript
function onChatChanged() {
    // 清除之前的队列和图片
    // 避免切换聊天后旧图片还在展示
    clearExistingImageQueue();
}
```

## 图片展示函数

### 1. 居中模态框弹窗

```javascript
/**
 * 在居中模态框中展示图片
 * @param {Object} image - 图片元数据
 */
function showImageModal(image) {
    const modal = document.getElementById('chat-images-modal');
    const modalImg = document.getElementById('chat-images-modal-img');

    if (!modal || !modalImg) return;

    // 设置图片
    modalImg.src = getImageUrl(image.filename);
    modalImg.alt = image.name || '聊天图片';

    // 显示模态框
    modal.classList.add('open');

    // 自动关闭
    if (currentSettings.modalAutoClose) {
        setTimeout(() => {
            modal.classList.remove('open');
        }, currentSettings.modalAutoCloseDelay);
    }
}

// 关闭模态框
function closeImageModal() {
    const modal = document.getElementById('chat-images-modal');
    if (modal) modal.classList.remove('open');
}
```

### 2. 插入到聊天

```javascript
/**
 * 将图片作为消息插入到聊天中
 * @param {Object} image - 图片元数据
 */
function insertImageToChat(image) {
    const { chat, addOneMessage, name1 } = SillyTavern.getContext();

    // 构建图片消息
    const imageMessage = {
        name: name1,
        is_user: true,
        is_system: false,
        send_date: Date.now(),
        mes: `[图片] ${image.name || ''}`,
        extra: {
            image: image.filename,
            type: 'chat-image',
        },
    };

    // 添加到聊天
    addOneMessage(imageMessage);

    // 或使用 HTML 图片标签
    const imageHtmlMessage = {
        name: name1,
        is_user: true,
        is_system: false,
        send_date: Date.now(),
        mes: `<img src="${getImageUrl(image.filename)}" alt="${image.name || '聊天图片'}" class="chat-image">`,
    };

    addOneMessage(imageHtmlMessage);
}
```

## 其他相关事件

除了核心的匹配触发事件外，还可以监听以下事件来完善插件功能：

### 设置变更时重新加载

```javascript
// 当设置更新时重新加载
eventSource.on(event_types.SETTINGS_UPDATED, () => {
    loadSettings();
});

// 聊天切换时重置状态
eventSource.on(event_types.CHAT_CHANGED, () => {
    closeImageModal();
});
```

### 生命周期钩子

```javascript
// 插件禁用时清理
export function onDisable() {
    closeImageModal();
    // 移除事件监听
    const { eventSource, event_types } = SillyTavern.getContext();
    eventSource.removeListener(event_types.MESSAGE_RECEIVED, onMessageReceived);
    eventSource.removeListener(event_types.MESSAGE_SWIPED, onMessageSwiped);
}

// 插件启用时重新注册
export function onEnable() {
    const { eventSource, event_types } = SillyTavern.getContext();
    eventSource.on(event_types.MESSAGE_RECEIVED, onMessageReceived);
    eventSource.on(event_types.MESSAGE_SWIPED, onMessageSwiped);
}

// 插件删除时清理数据
export async function onDelete() {
    // 清理 DOM 元素
    $('#chat-images-drawer').remove();
    $('#chat-images-modal').remove();
}
```

## 重要注意事项

1. **流式生成**: 流式生成时 `MESSAGE_RECEIVED` 在完整消息生成完成后才触发，所以无需处理中间 token
2. **重复匹配**: 确保同一消息不会触发多次匹配（可使用消息 ID 去重）
3. **异步处理**: 图片加载是异步的，确保在图片加载完成后再展示
4. **资源清理**: 模态框关闭时清理图片资源，避免内存泄漏
5. **错误容错**: 单个规则的正则错误不应影响其他规则的执行
