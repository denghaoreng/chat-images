# 图片管理

## 概述

图片管理是聊天图片插件的核心功能之一。用户可以通过 UI 上传图片，插件将图片复制存储到插件目录下的 `images/` 文件夹中，并配套生成 JSON 元数据来管理图片信息和权重。

### 图片文件命名

为避免文件名冲突，上传时自动生成唯一文件名：

```javascript
import { uuidv4 } from '../../../script.js';

/**
 * 生成唯一文件名
 * @param {string} originalName - 原始文件名
 * @returns {string} 唯一文件名
 */
function generateUniqueFilename(originalName) {
    const ext = originalName.split('.').pop(); // 获取扩展名
    const uniqueId = uuidv4().split('-')[0];    // 取 UUID 前一段
    return `img_${uniqueId}.${ext}`;
}
```

```

        │

        │
        ▼

        │

        │

        │

```

### 上传实现

```javascript
/**
 * 处理图片上传
 * @param {string} ruleId - 规则 ID
 */
async function handleImageUpload(ruleId) {
    // 创建文件输入元素
    const fileInput = document.createElement('input');
    fileInput.type = 'file';
    fileInput.accept = 'image/png,image/jpeg,image/gif,image/webp';
    fileInput.multiple = false;

    fileInput.onchange = async function(e) {
        const file = e.target.files[0];
        if (!file) return;

        // 验证文件类型
        const allowedTypes = ['image/png', 'image/jpeg', 'image/gif', 'image/webp'];
        if (!allowedTypes.includes(file.type)) {
            toastr.warning('仅支持 PNG、JPEG、GIF、WebP 格式的图片');
            return;
        }

        // 验证文件大小（5MB 限制）
        const maxSize = 5 * 1024 * 1024;
        if (file.size > maxSize) {
            toastr.warning('图片大小不能超过 5MB');
            return;
        }

        // 生成唯一文件名
        const filename = generateUniqueFilename(file.name);

        // 读取文件内容并保存
        const reader = new FileReader();
        reader.onload = async function(ev) {
            const fileData = ev.target.result; // Base64 data URL

            // 保存图片到服务器
            await saveImageFile(filename, fileData);

            // 创建图片元数据
            const imageMeta = {
                id: `img_${uuidv4().split('-')[0]}`,
                filename: filename,
                originalName: file.name,
                name: file.name.replace(/\.[^/.]+$/, ''),
                weight: 50,
                uploadDate: new Date().toISOString(),
                fileSize: file.size,
            };

            // 添加到规则
            addImageToRule(ruleId, imageMeta);

            toastr.success('图片上传成功');
        };
        reader.readAsDataURL(file);
    };

    fileInput.click();
}
```

## 图片保存（服务端通信）

前端通过 API 将图片文件发送到 SillyTavern 服务端保存：

```javascript
/**
 * 保存图片文件到插件目录
 * @param {string} filename - 文件名
 * @param {string} fileData - Base64 图片数据
 */
async function saveImageFile(filename, fileData) {
    const response = await fetch('/api/plugins/chat-images/upload', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ filename, data: fileData }),
    });

    if (!response.ok) {
        throw new Error('图片保存失败');
    }
}
```

### 服务端 API 端点

需要在 SillyTavern 的服务端注册对应的 API 路由来处理文件保存。

或者，可以使用更简单的方式——将图片数据以 Base64 形式存储在 data.json 中（适合小图片），但对于较大图片，建议直接写文件。

## 图片展示

### 缩略图列表

在规则编辑 UI 中，以缩略图网格展示绑定的图片：

```javascript
function renderRuleImages(rule) {
    const container = $(`#rule-${rule.id} .rule-images`);
    container.empty();

    for (const img of rule.images) {
        const imgUrl = getImageUrl(img.filename);

        const imgElement = $(`
            <div class="rule-image-item" data-image-id="${img.id}">
                <img class="rule-image-thumb" src="${imgUrl}" title="${img.name}">
                <div class="rule-image-weight">
                    <label>权重:</label>
                    <input type="range" min="0" max="100" value="${img.weight}"
                           class="image-weight-slider" data-image-id="${img.id}">
                    <span class="image-weight-value">${img.weight}</span>
                </div>
                <button class="rule-image-delete menu_button menu_button_icon"
                        data-image-id="${img.id}">
                    <i class="fa-solid fa-xmark"></i>
                </button>
            </div>
        `);

        container.append(imgElement);
    }
}

/**
 * 获取图片的访问 URL
 * @param {string} filename
 * @returns {string}
 */
function getImageUrl(filename) {
    return `/scripts/extensions/third-party/chat-images/images/${filename}`;
}
```

### 权重调整 UI

每张图片旁边显示一个滑块，用户可直观调整权重：

## 图片删除

```javascript
/**
 * 从规则中删除图片
 * @param {string} ruleId - 规则 ID
 * @param {string} imageId - 图片 ID
 */
async function deleteImage(ruleId, imageId) {
    const rules = getRules();
    const rule = rules.find(r => r.id === ruleId);
    if (!rule) return;

    // 从规则中移除图片
    const imageIndex = rule.images.findIndex(img => img.id === imageId);
    if (imageIndex === -1) return;

    const image = rule.images[imageIndex];
    rule.images.splice(imageIndex, 1);

    // 保存数据
    await saveRules(rules);

    // 重新渲染 UI
    renderRuleImages(rule);

    toastr.success(`图片 "${image.name}" 已删除`);
}
```

## 注意事项

1. **文件大小限制**: 建议限制单张图片不超过 5MB，避免加载过慢
2. **图片格式**: 建议支持 PNG、JPEG、GIF、WebP 四种常见格式
3. **文件名冲突**: 使用 UUID 生成唯一文件名，避免重名覆盖
4. **存储路径**: 确保 `images/` 目录在插件安装时自动创建
5. **缩略图缓存**: 可考虑生成缩略图（thumbnail）以加速列表渲染
6. **清理**: 在 `onDelete` 钩子中清理上传的图片文件
