## 🚀 快速开始

1. 插件位于 `third-party/chat-images/` 目录
2. 重启 SillyTavern 后，顶部导航栏会出现图片图标
3. 点击图标打开功能面板，添加正则规则和绑定图片
4. AI 回复后自动匹配并展示图片

## ✨ 核心特性

- **🎨 顶部导航栏入口** — 和世界树、酒馆设置一样的抽屉式按钮
- **🔍 正则匹配** — AI 回复后自动检测，支持多条规则
- **🖼️ 图片绑定** — 一个正则可绑定多张图片，支持权重概率
- **📤 图片上传** — 上传后复制到插件目录，配套 JSON 管理
- **🪟 多种展示** — 居中模态框弹窗 + 聊天消息插入
