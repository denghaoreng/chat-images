## 🚀 快速开始

1. 插件位于 `third-party/chat-images/` 目录
2. 重启 SillyTavern 后，顶部导航栏会出现图片图标
3. 点击图标打开功能面板，按需创建：**角色集 → 规则集 → 规则**
4. 为规则上传图片并设置权重，填写正则表达式
5. AI 回复后自动匹配并展示图片

## ✨ 核心特性

- **🎨 顶部导航栏入口** — 与酒馆原生UI风格一致的三标签抽屉面板
- **🔍 正则匹配** — 支持规则集分组、按序匹配、停留时间队列轮播
- **🖼️ 图片绑定** — 每张图片独立权重，加权随机选图
- **📤 批量操作** — 批量上传 + 批量创建规则
- **📱 移动端适配** — `100dvh` 全屏放大、点击时间戳检测双击
- **🪟 规则折叠记忆** — 折叠状态持久化、一键展开/折叠全部
